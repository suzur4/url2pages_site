# URL → ページ表示（情報Ⅰ・体験学習サイト）

「URLを入力してからページが表示されるまで」を、生徒がブラウザで操作して学ぶ教材です。
DNS（名前→住所の変換）／パケット（データの梱包）／ルーティング（経路選択）の3パートを、
反転学習＋ジグソー法で学べます。各パートに「体験・理解チェック・説明準備」があり、
最後に「全体の流れ（統合）」でつなげます。

## ファイル構成（GitHub Pages 用・静的サイト）
- `index.html` … 画面の構造
- `style.css` … 見た目（ライト/ダーク、文字サイズ、レイアウト）
- `script.js` … 動き（体験・クイズ・保存など）
- `.nojekyll` … GitHub の Jekyll 処理を無効化する空ファイル（そのまま置く）

## 保存・プライバシー
- 生徒の入力・進捗・テーマ設定は、その端末のブラウザ内（localStorage）にのみ保存されます（ログイン不要）。
- パスワードや個人を特定する情報は入力させない設計です（GitHub Pages も機微情報の送受信には使わない方針）。
